# fic-downloader-expanded

An extended Firefox plugin that adds automation on top of the basic fic downloader.

## Features

- Auto-download fics when you bookmark or kudos them
- History-saver mode: auto-download fics as you view them
- Route different triggers to different folders
- Downloads to custom filesystem folders via a native host companion app

## Requirements

- The native host companion app must be installed separately (handles filesystem access beyond Firefox's sandbox).
