# Fic Downloader

A suite of Firefox plugins for downloading fan fiction from popular sites.

## Plugins

### [fic-downloader](./fic-downloader/)

A standard Firefox plugin for manually downloading fics. Supports multiple output formats and works across major fic-hosting sites.

**Target sites (initial):** AO3, FanFiction.net, Royal Road

### [fic-downloader-expanded](./fic-downloader-expanded/)

An extended plugin with automation features: auto-download on bookmark, kudos, or page view (history-saver mode). Supports custom download folders per trigger type via a native host companion app.

> **Note:** Get the basic plugin working first. The expanded plugin builds on it.

## Planned

**fic-downloader-cli** — A command-line tool that can download fics to a chosen folder, defaulting to the current directory. Useful for scripting and automation outside the browser.
